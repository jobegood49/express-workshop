# Learn express

## How to run

Start:

```sh
npm start
#or
yarn start
```

Development:

```sh
npm run dev
#or
yarn dev
```

## REST API EndPOints

- `/`= Hello
- GET `/users`= List of users
- POST `/users`= Create new user
- DELETE `/users`= Delete all users
- DELETE `/users/:id`= Delete user by id
- PUT `/users`= Edit user by id
